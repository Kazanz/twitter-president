About

The Kazanski Poll is a realtime opinion mining suite that detects the overall sentiment of twitter users from every state in the USA.
The sofware analyzes tweets in realtime looking for mentions of the current predidential candidates.
When a match is detected the tweet is analyzed using state-of-the-art sentiment analysis algorithms.
The software ranks the overall sentiment of the tweet from 0 - 100 (0 being completely negative, 100 being completely positive) and detects the state where that user is located.
The overall sentiment for each state is then calculated by taking the mathematical average of all tweet's sentiments from that state.
A candidates overall national sentiment is also calculated by multiplying the average sentiment of each state by that state's number of tweets,
then the sums are added together and divided by the total number of tweets for that candidate. This produces the overall national sentiment towards a particular candidate.
By observing the highest sentiment in a particular state as well as the number of tweets one may accurately predict who will win elections in each of the states.
